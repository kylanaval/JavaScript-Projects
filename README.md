# JavaScript Projects

A collection of beginner-friendly JavaScript projects that demonstrate core JavaScript concepts, DOM manipulation, event handling, functions, loops, and user interaction.

These projects were created to practice and improve my front-end web development skills using **HTML**, **CSS**, and **JavaScript**.

---

## 📋 Projects

### 🍕 Pizza Menu
A pizza ordering application that allows users to customize their pizza by selecting different toppings. The application automatically calculates the total price based on the selected options.

**Features**
- Select pizza size
- Choose multiple toppings
- Automatic price calculation
- Interactive order summary
- User-friendly interface
---

### 🎮 Tic-Tac-Toe Game
A classic two-player Tic-Tac-Toe game built with JavaScript. The game randomly determines which player goes first, allows players to choose their markers, and celebrates the winner with animations and sound effects.

**Features**
- Two-player gameplay
- Random first player selection
- Player marker selection
- Win detection
- Tie game detection
- Winning animations and sounds
---
### 🧮 Calculator
A responsive calculator capable of performing basic arithmetic operations in a clean and modern interface.

**Features**
- Addition
- Subtraction
- Multiplication
- Division
- Clear function
- Responsive design
